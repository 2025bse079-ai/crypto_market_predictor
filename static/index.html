About this project

This system uses a Linear Regression model trained on Bitcoin historical data from 2020 to 2024, downloaded via Yahoo Finance. It predicts the next day's closing price based on 7 market indicators including moving averages, volume, and daily returns. Built with Python, Scikit-learn, Flask, and Chart.js.
